# FORAGE 按鈕素材說明

## 檔案位置
請將生成的 `forage_button.png` 放在此目錄下。

## 素材規格要求

### 尺寸
- **寬度**: 280px（或更高解析度，例如 560px @2x）
- **高度**: 80px（或更高解析度，例如 160px @2x）
- **格式**: PNG（帶透明背景）

### 視覺內容
根據設計圖，按鈕應包含：

1. **按鈕主體**:
   - 水平圓角矩形
   - 淺綠色到金黃色漸層
   - 光澤效果（頂部高光）
   - 金色邊框（3D 立體效果）

2. **文字 "FORAGE"**:
   - 位於按鈕左側
   - 深棕色，帶金色輪廓
   - 3D 立體效果

3. **購物車圖標**:
   - 位於按鈕右側，部分重疊
   - 白色車身，紅色把手
   - 可選：車內有採集物品（蘑菇、漿果等）

4. **背景**:
   - **完全透明**（Alpha 通道）
   - 不要添加任何背景色或邊框

## 使用方式

1. 將 PNG 圖片命名為 `forage_button.png`
2. 放在 `assets/images/` 目錄下
3. 組件會自動載入並使用

## 備註

- 建議使用 @2x 或 @3x 解析度以適配高解析度螢幕
- 確保背景完全透明，以便在不同背景下正確顯示
- 按鈕支援點擊回饋效果（縮放和透明度變化）
